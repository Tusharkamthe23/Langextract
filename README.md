# Langextract

Google LangExtract is an open-source Python library from Google that helps developers extract structured data from unstructured text using large language models (LLMs).

In simple terms, it lets you turn free-form text (emails, documents, chat logs, articles) into clean, structured outputs like JSON—without writing complex parsing rules.
